# Home-4 Page Extracted Project

This is a standalone Next.js project containing only the home-4 page and its dependencies.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000/home-4](http://localhost:3000/home-4)

## Structure

- src/app/ - Next.js app directory
- src/components/ - React components
- src/layouts/ - Headers and footers
- src/hooks/ - Custom hooks
- src/utils/ - Utility functions
- src/plugins/ - GSAP plugins
- public/assets/ - Static assets

## Key Technologies

- Next.js 16.0.1
- React 19.2.0
- GSAP 3.13.0
- Bootstrap 5.3.8
